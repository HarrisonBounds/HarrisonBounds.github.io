Color Quantization
with K-Means Clustering

Applying a lesser known K-Means variant, Jancey's Algorithm, to the Color Quantization problem

  • See Code

Color quantization (CQ) is a type of fixed-rate vector quantization developed for color images to reduce their number of distinct colors while keeping the resulting distortion to a minimum. Various clustering algorithms have been ap- plied to the CQ problem in the past four decades. Among these algorithms, hierarchical ones are generally more efficient (i.e., faster), while partitional ones are more effective (in minimizing distortion). Among the partitional algorithms, the effectiveness and efficiency of the Lloyd (or batch) k-means algorithm have been shown by multiple recent studies. In this paper, we investigate an alter- native, lesser-known k-means algorithm proposed by Jancey. Jancey k-means differs from Lloyd k-means only in updating the cluster centers at the end of each iteration. To obtain a competitive color quantizer, we develop a weighted version of Jancey k-means and then accelerate the weighted algorithm using the triangle inequality. Through extensive experiments on a diverse set of 100 color images, we demonstrate that, with the proposed modifications, Jancey k-means outperforms Lloyd k-means in both effectiveness and efficiency. In ad- dition, the proposed Jancey k-means-based color quantizer is as straightforward to implement as the popular Lloyd k-means color quantizer.
